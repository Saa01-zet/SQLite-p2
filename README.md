### Структура проекта
- **Selection/** - SELECT-запросы
- **Aliases/** - Псевдонимы (AS)
- **Filtering/** - Фильтрация, NULL и LIKE
- **Pagination/** - Пагинация (LIMIT и OFFSET)
- **Joins/** - Объединение таблиц (JOIN)
- **Aggregation/** - Агрегация и группировка
- **Logical_Order/** - Логический порядок и C#
- **Control_Task.sql** - Контрольное задание
